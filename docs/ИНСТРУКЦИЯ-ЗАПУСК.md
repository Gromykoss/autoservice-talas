# Инструкция по запуску и управлению проектом «Автосервис Талас»

> Для Сергея. Как управлять проектом через Hermes Agent с телефона (Telegram) или компьютера.

---

## Что нужно для старта

Уже готово:
- VPS сервер (тот же что у Алихана)
- Hermes Bridge :3000 (WhatsApp) — работает
- PostgreSQL — работает
- Домен: не нужен (WhatsApp)

---

## Этап 1: Подготовка WhatsApp

### 1. Получить новый номер WhatsApp

Нужен отдельный номер для автосервиса. Варианты:
- **SIM-карта:** купить местную симку в Таласе (~100 сом). Самый надёжный.
- **Виртуальный номер:** сервисы вроде SMS-activate (~$1-2).

### 2. Подключить номер к Hermes Bridge

Отправить мне в Telegram:
```
/автосервис подключи номер
```
Дальше я дам QR-код — отсканировать через WhatsApp Web на телефоне.

---

## Этап 2: Создание базы данных

Отправить в Telegram:
```
/автосервис настрой базу
```

Я создам таблицы для клиентов, машин и записей.

---

## Этап 3: Настройка бота

Отправить:
```
/автосервис настрой бота
```

Я создам main.py с базовыми сценариями:
- «запись» → диалог записи
- «список» → записи на завтра
- «отменить» → отмена записи
- «стоп» → передать диалог владельцу

---

## Этап 4: Запуск

Отправить:
```
/автосервис запуск
```

Бот начинает слушать WhatsApp. Первая неделя — тестовый режим: бот отвечает, но клиенты пока не подключены.

---

## Этап 5: Тестовый прогон

Проверяем на себе:
1. Пишем «запись» с тестового номера
2. Проверяем диалог
3. Смотрим «список»
4. Проверяем «отменить»
5. Пишем «стоп» — бот замолкает
6. Пишем «продолжить» — бот снова работает

Если что-то не так — пишем мне, правлю.

---

## Этап 6: Подключаем реальных клиентов

Когда всё отлажено — бот начинает отвечать реальным клиентам.

---

## Команды на каждый день (из Telegram)

| Команда | Что делает |
|---------|------------|
| `/автосервис статус` | Бот жив? Клиенты пишут? Ошибки? |
| `/автосервис записи` | Список записей на сегодня/завтра |
| `/автосервис логи` | Последние сообщения от клиентов |
| `/автосервис стоп` | Остановить бота |
| `/автосервис запуск` | Запустить после остановки |
| `/автосервис клиент ИМЯ ТЕЛЕФОН` | Добавить нового клиента |

---

## Что делать если

**Бот не отвечает клиентам:**
Написать `/автосервис статус` — я проверю и перезапущу.

**Бот ответил что-то не то:**
Написать «бот ошибся: клиент спросил X а бот ответил Y» — я поправлю сценарий.

**Нужно добавить новую фичу:**
Написать «/автосервис хочу чтобы бот спрашивал тип работ (масло/ходовая/кузов)» — добавлю.

**Клиент звонит а не пишет:**
Пока только WhatsApp. Если клиент звонит — вы отвечаете как обычно. В будущем можно автоответчик.

## Сколько это стоит

- Сервер: уже есть (0 сом дополнительно)
- DeepSeek API токены: ~$2-5/мес за все диалоги
- WhatsApp SIM: ~100 сом разово
- Настройка: разово, договоримся

Никаких ежемесячных подписок на софт.

- Сколько напоминаний отправлено
- Что можно улучшить

Если всё ок — идём на Этап 2 (запчасти).
